# Marketing-Campaign-Dashboard
A end-to-end marketing analytics project analyzing customer behavior, 
campaign performance, and spending patterns using Python and Streamlit.

## 🎯 Business Questions Answered
- Which customer segments spend the most?
- Which campaigns had the best response rates?
- What factors predict whether a customer responds to a campaign?

## 🛠 Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **Streamlit** — interactive dashboard
- **Jupyter Notebook** — analysis & EDA
- **Dataset** — Marketing Campaign Dataset (Kaggle)
